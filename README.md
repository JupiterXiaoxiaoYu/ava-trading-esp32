# ava-trading-esp32

Monorepo for the AVE trading ESP32 project.

Included components:
- `server/` - AVE backend, routing, trading logic, tests
- `simulator/` - PC simulator, mock scenes, screenshot verification
- `firmware/` - ESP32 firmware and board integration
- `shared/` - shared AVE UI screens used by firmware and simulator
- `docs/` - product, contract, and verification docs
- `config/` - shared config/assets used by the project
